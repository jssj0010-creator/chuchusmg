
# 킹즈마사지 정적 사이트 패키지 v2 (풀 메가메뉴 + 지역별 마이크로카피)

- 홈: `/`
- 지역: `/regions/index.html` (서울/경기/인천 및 세부 랜딩 전부)
- 공유 CSS: `/assets/style.css`

변경사항
- 상단 메가메뉴에 서울/경기/인천 전 세부 지역 링크를 모두 연결
- 지역/세부 랜딩 페이지에 지역 특성 기반 마이크로 카피 추가

배포
1) GitHub 리포 생성 후 전체 업로드
2) Netlify → New site from Git → 정적 사이트 (빌드 명령어 없음)
3) 도메인 `kingsmsg.com` 연결
